What is Video Streaming?
Streaming is a way of sending media files (like audio or video) in small parts over the internet. Instead
of downloading the whole file before playing, the client (browser or app) requests only the part it needs
to start playback.
In HTTP video streaming, the server handles range requests — it sends only specific byte ranges of
the file when the browser asks for them. This allows buffering, seeking, and efficient playback.
